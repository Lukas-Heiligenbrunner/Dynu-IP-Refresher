## Basic Build
Download source files

`git clone https://github.com/Lukas-Heiligenbrunner/DynuIPRefresher.git`

cd into downloaded files and Generate makefiles:

`cmake -S . -B build`

compile project

`make`

[root] install it to the system

`make install`

[root] or create Linux packages

`make package`
